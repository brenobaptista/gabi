# Gabi

> Gabi's professional portfolio.

## Features

- Next.js and all its features.
- 100% TypeScript.
- Tailwind CSS.
- ESLint + Prettier + Husky + Lint Staged.

## Getting Started

```
Install dependencies:
$ yarn

Serve with hot reload at localhost:3000
$ yarn dev
```
