# Gabi

> Gabi's professional portfolio made using Next.js + Tailwind CSS + TypeScript.

## Features

- Next.js.
- Tailwind CSS (light/dark theme).
- TypeScript.
- ESLint + Prettier + Husky + Lint Staged.

## Getting Started

```
Install dependencies:
$ yarn

Serve with hot reload at localhost:3000
$ yarn dev
```
